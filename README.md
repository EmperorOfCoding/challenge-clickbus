# challenge-clickbus

Projeto contendo o design system, os slides da apresentação e a documentação técnica para o desafio da Clickbus.

## Estrutura do Projeto

- `/design`: Contém o arquivo `design-system.html` e a apresentação visual `slides.html` e seu respectivo PDF `slides.pdf`.
- `/docs`: Contém a documentação sobre o projeto (`clickbus_projeto.pdf`, `clickbus_projeto.txt`) e ativos de marcação visual (SVG).

## Como Utilizar

- Para visualizar a apresentação de slides, abra o arquivo `design/slides.html` em qualquer navegador moderno.
- Para verificar os componentes do Design System criado, abra o arquivo `design/design-system.html` em um navegador.
- Os documentos detalhados sobre o projeto podem ser encontrados na pasta `/docs`.
